###########
Development
###########

Resources for contributing to and extending lcviz.

See the `GitHub repository <https://github.com/spacetelescope/lcviz>`_ and
`CONTRIBUTING.md <https://github.com/spacetelescope/lcviz/blob/main/CONTRIBUTING.md>`_
for contribution guidelines and development setup.

lcviz is built as a jdaviz extension. See the
`jdaviz development docs <https://jdaviz.readthedocs.io/en/latest/dev/index.html>`_
for guidance on extending jdaviz.
